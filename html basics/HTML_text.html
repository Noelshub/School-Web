Formatera text med HTML

      Allt vi skriver
      inom pre-taggar 
      behåller
      sin formatering

Text-level tags

      small lvl -1 - t.ex. copyright, marginalteckning
      bold lvl 0 - mest visuellt
      italics lvl 1 - t.ex. tekniska termer
      mark lvl 1,5 - highligtad text, kanske refereras till
      emphasis lvl 2 - betoning
      strong lvl 3 - STARK betoning, mycket viktigt

citationer

cite titel på verk - t.ex. bok inline citation, kortare citat, kan ha attributet cite för att länka

block quote används för längre citat

Förklarnde text-tags

      dfn - används när vi använder en defitiniton som förklarars i kontexten
      abbr - när vi har en förkortning
