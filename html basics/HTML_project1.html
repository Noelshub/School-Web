Vad är HTML och vad används det för?

HTML står för HyperText Markup Language och används för att skapa strukturen och innehållet på webbsidor. Det är ett märkspråk som använder taggar för att definiera olika delar av en sida.

Vad är XML och XHTML?

XML är mer flexibelt än HTML, möjlighet att själv definiera datastruktur och element, används ofta för data överföring. XHTML är mer strikt än HTML, man måste använda korrekt syntax för att det ska kunnas tolkas.
Mer konsekvent hantering av element vilket kan leda till mer strukturerad och mer lättförståelig kod.
