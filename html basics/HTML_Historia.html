Vad är World Wide Web och vem var dess skapare?

WWW är ett nätverk, skapat av Tim Burners-Lee. Tim gillade standardiseringar, och skapade därför protokoll för bland annat HTML

Vilka teknologier introducerades under webbens tidiga utveckling för att strukturera och designa webbsidor?

  • HTML
  • CSS
  • Java Script

Vad innebär begreppet "responsiv design" när det gäller webbutveckling?

Hemsidan ska vara mer anpassad 
till användare som t.ex. Viewport som anpassar hemsidan till användarens enhet.

Hur har ökningen av mobila enheter påverkat webbutveckling?

Det har populariserat responsiv design och webbapplikationer eftersom att många använder behöver responsiv design

Vad är en webbapplikation? Ge exempel på några existerande webbapplikationer.

Webbapplikationer är applikationer som man kan använda på en hemsida utan att installera något program som t.ex. Gmail, Facebook och Amazon mm.
