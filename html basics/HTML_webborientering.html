HTML webborientering

Vad är en webbläsare och vad gör den?

Mjukvara (program), används för att visa och integrera med webbsidor och webbapplikationer på internet. Skickar HTTP-förfrågan till webbserver. Exempel: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari och Opera.

Vad är en webbserver och vad gör den?

En dator eller en mjukvara som lagrar, hanterar och levererar webbsidor och andra webbresurser till använder via internet. Skickar HTML-, CSS- och JavaScript-kod till webbläsare.

Vad är en URL och vad innehåller den?

URL (Uniform Resource Locator), identifiera och hitta webbplatser. Består av protokoll prefix (ex http, https) och domännamn ( ex. google.com) och ibland sökväg till specifik resurs ( …/project/example2)

Vad är skillnaden mellan HTTP och HTTPS?

HTTP är inte krypterat medan HTTPS är krypterat så det är säkrare. HTTP ( HyperText Transfer Protocol) är det vanligaste protokollet för att överföra data mellan webbläsare och webbserver. HTTPS (HTTP Secure) : säker version av HTTP där data krypteras för att skydda användarinformation och förhindrar obehörig åtkomst. Använder SSL/TLS.
