CSS kunskap

Vad är CSS? Hur används det?

Cascading Style Sheets. Används för att stilisera HTML-kod, för att styra utseendet och layouten av webbsidor.

Vad är skillnaden mellan inline-stil, inbäddad stil och extern stil i CSS?

Inbäddad: med style taggar används oftare när det är bara det dokumentet som har den här stylen.

Inline t.ex. p style=””blah blah p. Extern betyder att det i ett separat dokument används när stilen appliceras på mer än ett html-dokument.

Vad är en selektor i CSS och hur används den?

Det finns fem olika sorter. Enkla: väljer element utifrån namn, id, klass.

Kombinerade: väljer ett element utifrån förhållande mellan element t.ex. Child till något parent element.

Pseudo Klass: väljer utifrån vad som “händer” med ett element t.ex. Hover.

Pseudoelement: väljer del av element t.ex. Första bokstaven/linjen i p. Attribut: väljer utifrån angivet attribut på element.

Vad är skillnaden mellan klass-selektor och ID-selektor i CSS?

Klass kan ge till flera element. Medan id ska man bara ge till ett element.(man kan ge till flera men det blir problem)

Vad är kaskadordningen i CSS och hur påverkar den hur stilar tillämpas på element?

1.Arv - Om element saknar regel så söks en “förälders” stilregel och appliceras.

2. Relevans - Behåller enbart regler som matchar element.

3. Ursprung och betydelse - Sorterar regler baserat på hur viktiga de är och vem som har applicerat dem.

4. Noggrannhet - väljer regler med högre specificitet om relevanta regler har samma ursprung och betydelse.

5. Ordning - Om specifika regler har samma ursprung och betydelse väljer kaskadmodellen det senaste angivna värdet i källkoden.
