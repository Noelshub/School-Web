Vad är XML och XHTML?

XML är mer flexibelt än HTML, möjlighet att själv definiera datastruktur och element, används ofta för data överföring. XHTML är mer strikt än HTML, man måste använda korrekt syntax för att det ska kunnas tolkas. Mer konsekvent hantering av element vilket kan leda till mer strukturerad och mer lättförståelig kod.

Vad är DOM, och varför är den viktig?

Document object model, representation över hemsidans struktur, med en trädformad hierarki och det är särskilt viktigt att använda DOM ifall klienten ska kunna manipulera sidan.
